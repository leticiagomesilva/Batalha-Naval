# Batalha Naval

Jogo de **Batalha Naval** desenvolvido com **HTML, CSS e JavaScript puro**, com foco em manipulação do DOM, eventos, renderização dinâmica e organização em múltiplas fases de jogo.

O projeto foi construído para funcionar com **dois jogadores humanos na mesma tela**, seguindo uma estrutura separada entre arquivos de marcação, estilo e lógica.

## Como funciona

A partida acontece em etapas:

1. Na tela inicial, os jogadores informam seus nomes.
2. Cada jogador posiciona sua frota no tabuleiro.
3. O jogo alterna os turnos entre os dois participantes.
4. Cada jogador ataca coordenadas do tabuleiro inimigo.
5. Vence quem afundar todos os navios adversários primeiro.

## Recursos do jogo

- Tela inicial para configuração dos jogadores
- Alternância de turnos entre os jogadores
- Tabuleiro de ataque e defesa
- Placar atualizado dinamicamente
- Histórico de tiros em tabela dinâmica
- Destaque visual para o jogador da vez
- Tela final com resultado da partida

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
